# Slidev 主题排版与布局优化设计

**日期**: 2026-05-09
**作者**: @Echo009
**方案**: B — 布局 + 排版双优化

## 背景

当前 Slidev 赛博朋克主题视觉风格已建立，但存在排版和可读性问题：文字层次不够清晰、内容密度高时难以快速扫视、布局空间利用不够合理。本次优化在不改变赛博朋克风格的前提下，精调排版系统和布局间距。

## 1. 全局排版系统（index.css）

### 字体尺寸比例

| 元素 | 当前 | 调整后 | 理由 |
|------|------|--------|------|
| h1 | 2.8rem | 2.6rem | 内容页 h1 收敛，封面页单独放大 |
| h2 | 1.8rem | 1.5rem | 与 h1 拉开层次差距 |
| h3 | 1.3rem | 1.15rem | 作为内容页主力标题，不抢视觉 |
| 正文 | 1rem | 1.05rem | 略增大提升可读性 |
| 行高 | 1.8 | 1.85 | 适当放松行间距 |
| 字间距 | 0.01em | 0.015em | 微调让文字不挤 |

### 色彩对比度

- 正文颜色: `#d0d0d8` → `#d8d8e0`
- 次要文字: `rgba(224,224,224,0.55)` → `rgba(224,224,224,0.7)`
- h2 下划线渐变增强可见度

### 代码块

- 内边距: `1.2rem` → `1.4rem`
- 内联代码字号: `0.88em` → `0.92em`
- 顶部渐变条更醒目

## 2. 布局组件间距（default / two-col / cards）

### 共同调整

| 属性 | 当前 | 调整后 |
|------|------|--------|
| 侧边栏宽度 | 180px | 140px |
| 整体 padding | `3rem 4rem 4rem` | `2.5rem 3.5rem 4.5rem` |

### default.vue

- 点阵背景间距: `24px` → `28px`
- HUD 顶部发光高度: `20px` → `12px`
- 竖线装饰透明度降低
- 底部进度条: `1px` → `2px`

### two-col.vue

- 双栏间距 gap: `1.5rem` → `2rem`
- 中间分割线渐变更柔和

### cards.vue

- 卡片内 padding: `1.2rem 1.4rem` → `1rem 1.2rem`
- 卡片间 gap: `1.5rem` → `1.2rem`

## 3. 封面页精调（cover.vue）

### 标题区

| 属性 | 当前 | 调整后 |
|------|------|--------|
| h1 字号 | 2.8rem | 3.2rem |
| cover-tag 字号 | 0.7rem | 0.6rem |
| cover-info 字号 | 1.15rem | 1rem |
| cover-info 颜色 | `#ff006e` | `#ff006ecc` |
| 装饰横线宽度 | 180px | 240px |

### 装饰元素

- 旋转圆环: `340px/420px` → `280px/360px`，透明度 `0.12` → `0.08`
- 四角装饰: `24px` → `18px`

### 底部状态栏

- 字号: `0.55rem` → `0.5rem`
- 状态文字颜色降低透明度

### 移动扫描线

- 高度: `80px` → `60px`，速度: `8s` → `6s`

## 涉及文件

| 文件 | 改动类型 |
|------|----------|
| `slides-source/theme/styles/index.css` | 排版系统 + 色彩 + 代码块 |
| `slides-source/theme/layouts/default.vue` | 间距 + 侧边栏 + 背景 + 进度条 |
| `slides-source/theme/layouts/two-col.vue` | 间距 + 侧边栏 + 双栏 gap |
| `slides-source/theme/layouts/cards.vue` | 间距 + 侧边栏 + 卡片 |
| `slides-source/theme/layouts/cover.vue` | 标题 + 装饰 + 状态栏 |

## 不涉及的文件

- `section-header.vue` — 不动
- `intro.vue` — 不动
- `index.ts` — 不动

## 改动原则

- 所有改动均为 CSS/间距调整，不改变组件结构和功能
- 保持赛博朋克风格一致性
- 不引入新的装饰元素或动画
